# ProVAI UI Style Guide

This document defines the specific visual and linguistic standards for the ProVAI application, ensuring a cohesive and professional brand identity.

## 1. Color Palette

We use a calm, low-contrast palette inspired by the GitHub Dark Dimmed theme to promote focus and serenity.

| Role | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Background** | `#0d1117` | The main application background. |
| **Secondary Background**| `#161b22` | For containers, sidebars, and chat bubbles. |
| **Border Color** | `#30363d` | For borders on components and inputs. |
| **Primary Text** | `#adbac7` | For all standard body text and labels. |
| **Accent Interactive** | `#58a6ff` | For focus glows, selected tabs, and key links. |
| **Accent Success** | `#238636` | For primary action buttons and success messages. |

## 2. Typography

We use the "Inter" font for its exceptional clarity and readability.

| Element | Font Family | Weight | Size (Relative) | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Page Title** (`st.title`) | `Inter` | `700` (Bold) | `2.5rem` | |
| **Header** (`st.header`)| `Inter` | `600` (Semi-Bold)| `1.75rem`| |
| **Subheader** (`st.subheader`) | `Inter` | `600` (Semi-Bold)|`1.25rem` | |
| **Body Text** | `Inter` | `400` (Regular) | `1rem` (base) | |

## 3. Spacing

We use a base unit of **8px**. All margins, paddings, and gaps should be multiples of this unit (e.g., 8px, 16px, 24px) to create a consistent visual rhythm.

## 4. Tone of Voice

The language of ProVAI must always reflect its role as a "Personalized AI Tutor."

-   **Patient & Encouraging:** Use positive reinforcement. Frame errors as opportunities to try again. (e.g., "That's a great question, let's break it down.")
-   **Clear & Concise:** Avoid technical jargon. Use simple, direct language.
-   **Professional & Respectful:** The AI is a tutor, not a casual friend. The tone should be helpful and professional, maintaining a respectful distance.
